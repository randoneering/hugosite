Randoneering, LLC

Database Engineering

I optimize and maintain database systems with focus on PostgreSQL and other relational databases. My work includes diagnosing and resolving query performance bottlenecks, tuning engine parameters for specific workloads, and automating deployment pipelines. I implement maintenance schedules, enforce schema design standards, and establish monitoring systems that identify issues before they impact production.

Data Engineering

I build and maintain data pipelines using Python and enterprise SaaS platforms including Rivery, Airbyte, and AWS Glue. For Snowflake environments, I handle account administration, implement cost controls through warehouse optimization and query analysis, and enforce role-based security policies. I architect data product pipelines from ingestion through transformation, implementing validation checks that maintain data quality throughout the pipeline.

Engagement Model

I work with teams that need database performance resolved, data pipelines stabilized, or data infrastructure costs reduced. Projects typically involve inherited databases requiring optimization, new data pipeline development, or establishing practices that prevent data quality issues and technical debt from accumulating.
